# Upgrade to Silverstripe CMS 6

This document outlines the key changes required to upgrade to the Silverstripe CMS 6 compatible version of this module.

## Core Dependency Updates

⚠️ **BREAKING CHANGE:** The core Silverstripe CMS dependencies have been upgraded to the next major version. You will need to ensure your project is compatible with Silverstripe CMS 6 before upgrading this module.

-   `silverstripe/framework` has been upgraded from `^5.0` to `^6.0`.
-   `silverstripe/admin` has been upgraded from `^2.0` to `^3.0`.
